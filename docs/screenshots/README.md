## How to Add Screenshots

1. **Take screenshots** using your OS's screenshot tool or browser
2. **Save** them in this folder with the naming convention above
3. **Update** the README.md to reference them:
   ```markdown
   ![Login Page](./docs/screenshots/01-login.png)
   ```

## Tips for Good Screenshots

- Use a clean browser window (no extensions cluttering)
- Show both desktop and mobile views if responsive
- Highlight key features with arrows or circles in image editor
- Keep file size small (compress if needed)
